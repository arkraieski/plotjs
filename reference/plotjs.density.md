# plotjs Method for Kernel Density Estimation

The `plotjs` method for density objects.

## Usage

``` r
# S3 method for class 'density'
plotjs(x, main = NULL, xlab = NULL, ylab = "Density", type = "l", ...)
```

## Arguments

- x:

  a "density" object

- main:

  a main title for the plot.

- xlab:

  label for the x axis

- ylab:

  a label for the y axis, defaults to a description of y.

- type:

  1-character string giving the type of plot desired. The following
  values are possible: "p" for points, "l" for lines, and "b" for both
  points and lines.

- ...:

  arguments passed to other methods
